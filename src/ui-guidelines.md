# UI Implementation Guidelines

These are working guidelines for the SkillBound web app while a formal UI/UX design is not yet available.

- Keep border radius minimal across the application.
- Do not apply radius on very small elements such as dropdown options, tool menu items, or compact buttons.
- Prefer rectangular blocks for menus and small controls.
- Use consistent spacing and compact padding for controls.
- Keep the visual language simple and functional.
- Use the existing theme palette consistently (marshland / reef-gold).
- Favor clear content hierarchy with strong headings and readable body text.
- Avoid heavy visual decoration; focus on clarity and ease of use.
